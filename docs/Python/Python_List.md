# Python List

1. list.index(a) 的时间复杂度是O(n)， 从头往后搜索，返回index。
2. list[:] 是常见的list copy，list[:] 和 list[a:b] 都会创建一个新的list。
3. list.pop() 移除并返回最后一个，时间复杂度O(1)；list.pop(0) 移除并返回第一个，然后将后面的元素全部往前移了一格，时间复杂度O(n).
