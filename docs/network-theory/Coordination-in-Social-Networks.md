### Plain Networked Coordination Games
1. If player's action “match”, i.e. they play the same strategy,they get the same positive utility $$Q(X, X) = x$$ or $$Q(Y, Y ) = y$$
2. If they “mismatch”, they get utility $$Q(X, Y ) = Q(Y, Y ) = 0$$
